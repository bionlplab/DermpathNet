Lymphomatoid Papulosis Type A: A Case Report of the "Wait-and-See Strategy" in a 27-Year-Old Male Patient with Extensive Disease.

Histology revealed a diffuse lymphocytic infiltration in the reticular dermis by small, medium, and large sized lymphocytes, some of which with anaplastic nuclei, accompanied by a variable number of neutrophils and histiocytes (hematoxylin-eosin stain x 20 (a), x 40 (b), x 60 (c)).

PMID: 31380123

Type: Premalignant/Malignant, Category: Hematolymphoid
