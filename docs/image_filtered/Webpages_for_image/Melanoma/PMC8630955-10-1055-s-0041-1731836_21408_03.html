A Large Oral Melanoma: A Case Report of a Rare but Aggressive Malignancy.

Histopathologic features. ( A , B ) Hematoxylin and eosin stained section demonstrated several atypical, pleomorphic melanocytes with melanin pigments in basal and some suprabasal cell layers of the epithelium. A pagetoid spread of some melanocytes to the superficial layer of epithelium is observed ( A : ×100, B : ×200). ( C , D ) S-100 ( C : ×100, D : ×200). ( E , F ) HMB-45 ( E : ×100, F : ×200). ( G , H ) Melan-A ( G : ×100, H : ×200).

PMID: 34428837

Type: Premalignant/Malignant, Category: Melanocytic
