An open source automated tumor infiltrating lymphocyte algorithm for prognosis in melanoma.

Representative picture of a sample melanoma case showing the H&E image (Zoom: ×20, a) and the digital-image analysis (DIA) mask (b). Scale bar represents 20 µm. Using the NN192 algorithm, segmentation shows red indicates tumor cells, purple marks immune cells, green corresponds to stromal cells, and yellow indicate others (false cell detections or unknown or background). Since stromal and “other” cells are rare, large arrows are included to show example cells.
